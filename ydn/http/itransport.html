Class ydn.http.ITransport

Providing HTTP request.

Constructor
ydn.http.ITransport()
Create a new HTTP transport object.
Generally subclass are used and obtained from the Application object. The following subclass are available:
  • AccessTokenTransport
  • BearerAuthTransport
  • FilteredTransport
  • GseTransport
  • InterceptableTransport
  • JsonpTransport
  • ProxyTransport
  • XMLHttpRequestTransport
Methods
send(url, callback, options)
Send HTTP request.
Parameters:
{string} url
URL to send.
{function(ydn.http.CallbackResult)=} Optional.callback
Callback to be invoked when receiving the result.
{Object=} Optional.options
HTTP Options.
Returns:
{goog.async.Deferred|undefined}Result in deferred object, if callback is not provided.
