Class ydn.api.UserAssess

Providing authorization tokens.

Constructor
ydn.api.UserAssess()
Create a user assess object.
API user do not initiate user object, but acquired from application instance using getUser() method. 
var app = new ydn.api.App(options);
  var user = app.getUserAssess();
Events
{string} tokens-loaded
This class dispatch goog.events.Event of type tokens-loaded, when it received all tokens accessible to the login user.
{string} user-token-loaded
This class dispatch goog.events.Event of type user-token-loaded, when it received token belong to user.
{string} acls-loaded
This class dispatch goog.events.Event of type acls-loaded, when it received all ACL.

The same handler can only be added once per the type. Even if you add the same handler multiple times using the same type then it will only be called once when the event is dispatched.
