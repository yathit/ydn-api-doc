Class ydn.gdata.Entry

Class Feed represents Atom Entry object.

Constructor
ydn.gdata.Entry(feed)
Create a new Entry.

Parameters:
{string|GDataFeed=} feed
Feed URI or Feed content in JSON format.
Methods
copy(entry)
Copy data from given entry.
Parameters:
{!ydn.gdata.Entry} entry
Entry data to be copied.
Returns:
{boolean} true if copied.
deleteEntry()
Mark to delete an entry. Commit the gdata feed to complete the deletion.
getData()
Get entry data in JSON format.
Returns:
{!Object} data in JSON object.
getEntryId()
Get entry feed URI.
Returns:
{string} entry feed ID
getSingleId()
Get entry by single word entry id.
Returns:
{string} id in single word.
getEntryStatus()
Get entry status.
Entry status could be 'delete', 'deleted', 'gd:deleted', 'modified' or 'committed'.
Returns:
{string} status.
getKind()
Return kind of feed.
See common kind elements for more detail.
Returns:
{string} feed kind.
getType()
Get type of feed.
Currently this could be either or 'm8' or 'site'.
Returns:
{string} feed type.
hasDeleted()
Get deleted status.
Returns:
{boolean} true if the entry was marked for deletion.
isDeleted()
Get gd deleted status.
Returns:
{boolean} true if the entry is deleted in server.
isDirty()
Get data modification status.
Returns:
{boolean} true if the entry is modified but not received commited status from the server.
merge(entry)
Update data from given entry.
Parameters:
{!ydn.gdata.Entry} entry
Entry to merge with.
Returns:
{boolean} true if updated.
setTitle(title)
Set title.
Parameters:
{string} title
Title.
